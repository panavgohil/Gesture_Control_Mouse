# Gesture Control Mouse

A computer vision project that allows controlling the mouse using hand gestures.

## Features

- Mouse Movement
- Left Click
- Right Click
- Scrolling
- Volume Control
- Brightness Control

## Technologies Used

- Python
- OpenCV
- MediaPipe
- PyAutoGUI
- Pycaw
- Screen Brightness Control

## How It Works

The webcam tracks hand landmarks using MediaPipe. Different hand gestures are mapped to mouse and system controls.
